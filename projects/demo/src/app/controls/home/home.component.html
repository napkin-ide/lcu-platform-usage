Welcome to the LCU-Starter-App

LCU-Starter-App is an Angular Application built and customized by the lcu-cli - a implementation of the Angular CLI by Fathym.

This starter application provides the baseline library and application structure giving developers the freedom to rapidly create data-driven applications. It is the first step in Fathym's best practice workflow to build custom Low-Code Units (LCUs) tailored to your business requirements.

Read through our Documentation to get started on building your very own Low-Code Unit™!
